# TechHaven - Technology Solutions Platform

A fully responsive multi-page web application showcasing technology services.

## Project Overview
TechHaven is a professional website for a technology solutions company, featuring:
- 7+ interlinked pages
- Responsive design using Bootstrap 5
- Dark mode toggle
- Form validation
- Interactive gallery

## Pages
1. **Home** - Company overview and features
2. **About** - Company background and team
3. **Services** - IT solutions offered
4. **Blog** - Technology articles and news
5. **Gallery** - Project portfolio
6. **FAQ** - Frequently asked questions
7. **Contact** - Get in touch with us

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5

## Setup Instructions
1. Clone the repository
2. Open index.html in your browser
3. No build tools required - it's pure HTML/CSS/JS!
